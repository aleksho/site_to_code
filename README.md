# site_to_code

Репозиторий для хранения и обработки экспортированных дизайн-ресурсов из Figma проекта **«Cosmetology Egorova»** — сайт для образовательной платформы в сфере косметологии (Егорова Марина).

---

## 📋 Общая информация

- **Название файла в Figma:** `Cosmetology Egorova web (Copy)`
- **Страницы в макете:**
  - `desktop` (id: `0:1`)
  - `mobile` (id: `221:435`)
- **Цель:** Автоматизированный экспорт дизайн-токенов, стилей, ассетов и структуры страниц для последующей вёрстки и интеграции в Django-проект.

## 🎨 Дизайн-система

### Шрифты
- **Manrope** — основной текстовый шрифт (Regular 400, Medium 500, SemiBold 600, Bold 700)
- **Onest** — заголовочный шрифт (Regular 400, SemiBold 600, Bold 700)

### Палитра (основные цвета)
| Цвет | HEX | Назначение |
|------|-----|------------|
| Чёрный | `#000000` | Основной текст |
| Тёмно-серый | `#1E1E1E`, `#222222`, `#242424` | Заголовки, акценты |
| Серый | `#414141`, `#454545`, `#525252`, `#545454` | Вторичный текст |
| Средний серый | `#868686`, `#8A8A8A`, `#A1A1A1`, `#A7A7A7` | Плейсхолдеры, иконки |
| Светло-серый | `#BBBBBB`, `#BEBEBE`, `#D9D9D9`, `#E0E0E0` | Разделители, фоны |
| Тёплый беж | `#AAA195`, `#ADA098`, `#DCD8D3`, `#E9E2D7`, `#F3EDE5` | Фоновые блоки, акценты |
| Белый | `#FFFFFF` | Основной фон |

### Design Tokens
- **18 цветовых токенов** (`estno.ru/color/*`) — оттенки серого, чёрный, белый, тёплые бежевые
- **40 числовых токенов** — stroke weight, font size, font weight, line height, opacity

### Стили
- **21 стиль заливки** (Fill styles) — именованные цвета из палитры
- **27 текстовых стилей** — комбинации шрифтов, размеров и межстрочных интервалов
- **Семантические стили** — Button, Heading 1/2/3, Input, Item, Link, Strikethrough

## 🚀 План вёрстки (из отчёта)

1. **Переменные** — CSS custom properties из `variables.json`
2. **Типографика** — подключение `Manrope` + `Onest`
3. **Сетка** — Frame → `<div>`, bounds → width/height/position
4. **Компоненты** — кнопки, карточки, инпуты из semantic styles
5. **Адаптив** — Mobile (390px) → Desktop breakpoints
6. **Полировка** — border-radius, stroke-weight, opacity из `styles.json`
